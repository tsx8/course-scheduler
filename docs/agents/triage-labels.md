# Triage Labels

工程技能使用五个标准 triage roles。当前仓库未发现已有 label 约定，默认直接使用 role 名作为 GitHub label 名称。

| Skill role | GitHub label | Meaning |
|---|---|---|
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已充分说明，适合 agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会处理 |

如果仓库后来建立了不同 label 体系，优先修改右列，不要让技能创建重复语义的 labels。
