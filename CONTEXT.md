# CONTEXT.md

## Language

**课表版本**:
一套可独立切换、保存和导出的排课结果。课表版本共享教师、课程、校区、场地、时间段、工作日和约束配置；版本内只承载排课记录及其锁定、暂存等排课状态。
_Avoid_: 数据版本、数据库版本、备份

**基础数据**:
所有课表版本共享的排课前提，包括教师、课程、校区、场地、时间段、工作日、教师不可用时间、排课密度、教师-课程、课程-场地、教师-校区关系和已保存的校区筛选视图。基础数据不是某个课表版本的副本。
_Avoid_: 版本数据、数据副本

**当前课表版本**:
用户正在查看和编辑的课表版本。校区总课表、教师个人课表、问题检查、自动排课和 CSV 导出都默认作用于当前课表版本。
_Avoid_: 当前数据、激活数据

**排课记录**:
一次课程安排，连接教师、课程、校区、场地、工作日和时间段，并带有锁定、暂存等状态。
_Avoid_: 课程卡片（只用于 UI 展示）

**暂存课程**:
从课表中临时移出的排课记录。暂存课程属于当前课表版本，不参与自动排课，但切换、保存和回滚时必须保持一致。
_Avoid_: 草稿课程、待排课程

**校区总课表**:
以校区为主视角查看和编辑当前课表版本的排课结果。一个时间段内可以有多条排课记录。
_Avoid_: 校区视图（只在讨论 UI 文件时使用）

**教师个人课表**:
以教师为主视角查看和编辑当前课表版本的排课结果。它同时展示教师不可用时间段。
_Avoid_: 教师视图（只在讨论 UI 文件时使用）

## Relationships

- 一个课表版本包含多条排课记录。
- 当前课表版本是课表版本集合中的一个。
- 基础数据由所有课表版本共享。
- 校区总课表和教师个人课表都是当前课表版本的不同视角。
- 暂存课程是排课记录的一种编辑状态，不是独立课程实体。
